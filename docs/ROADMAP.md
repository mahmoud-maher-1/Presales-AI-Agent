# Product Roadmap

Development roadmap for **AI Tawasol**.

The roadmap reflects the evolution of the system from a backend foundation into a complete AI pre-sales platform.

---

## Phase 1 — Backend Foundation

Build the technical backbone of the application.

### Goals

- FastAPI backend
- PostgreSQL integration
- SQLAlchemy setup
- Docker environment
- API endpoint structure
- conversation and message persistence
- project and requirement models

### Outcome

A stable backend foundation ready for AI integration.

---

## Phase 2 — AI Integration

Connect the application to LLM providers.

### Goals

- Gemini API integration
- unified LLM service abstraction
- response generation
- prompt orchestration
- provider switching support

### Outcome

The system can generate intelligent conversational replies.

---

## Phase 3 — Pre-Sales Intelligence

Turn the assistant into an AI pre-sales engineer.

### Goals

- requirement extraction
- requirement memory update
- missing field detection
- next-question selection
- conversation guidance logic

### Outcome

The AI can conduct structured requirement discovery instead of only replying conversationally.

---

## Phase 4 — Project Structuring

Convert conversations into reusable business outputs.

### Goals

- structured requirement persistence
- project summary generation
- lead scoring
- lead status classification
- project creation from conversations

### Outcome

The system can transform raw conversations into structured sales and delivery inputs.

---

## Phase 5 — AI-Generated Presales Artifacts

Generate useful presales deliverables automatically.

### Goals

- SWOT analysis generation
- Kano classification
- activity diagram generation
- SRS draft generation
- PDF export support

### Outcome

The system produces business-ready presales documents.

---

## Phase 6 — Integrations

Expand communication channels and usability.

### Goals

- web chat integration
- Telegram integration
- WhatsApp integration
- CRM or internal workflow integration

### Outcome

The AI pre-sales agent becomes accessible across multiple channels.

---

## Phase 7 — Product Expansion

Turn the backend engine into a complete product platform.

### Goals

- dashboard
- analytics
- admin tools
- conversation monitoring
- project pipeline visualization
- team collaboration features

### Outcome

A full AI-powered presales operations platform.
